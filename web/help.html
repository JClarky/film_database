Overview

The software is used to manage a film database. You can view, add and edit movies using this software.

Main Dashboard

The main dashboard displays all movies. You can edit and delete each movie using the labelled buttons adjacent to the targeted movie. Navigate to the dashboard by clicking "Home" in the navbar.

Add movie

To add a movie, click "Add Film" on the main dashboard, or "Add Film" in the navbar. You can then add a movie by entering all details about it, then clicking the "Add Film" button.

Edit movie

To edit a movie visit the main dashboard by clicking "Home" in the navbar. Find the target movie and click "Edit". Once in the edit page, change the desired values then save the changes by clicking "Save".
